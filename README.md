# LeafLine

A terminal-based productivity timer that grows a forest of emoji trees and flowers as you focus on your tasks.

## Installation

```
pip install leafline
```

## Usage

Run the timer:

```
leafline
```

Follow the prompts to set your focus time and watch your forest grow!
